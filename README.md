# Moving Squares Demo

A simple Rust application demonstrating moving squares that wrap around the screen.

## Running

Make sure you have Rust installed, then run:

```bash
cargo run
```

## What it does

- Opens a window with a black background
- Displays 5 colored squares moving from left to right
- Squares wrap around to the left side when they reach the right edge
- Each square has different size, speed, and color
